# WebTestingHub — Root Files

This package contains the root pages and configuration entry points for the static, client‑side campaign system.

## Files
- `index.html` — Entry with flow chooser.
- `engagement.html` — Entry page for the Engagement Flow.
- `premium.html` — Entry page for the Premium Flow.
- `scripts/config.js` — Centralized config (OFFER_URL, limits).

## Notes
- Include `/assets` from ZIP 13 and `/scripts` from ZIP 11 at the same project root.
- Update `scripts/config.js` only if you need to change the OFFER_URL or limits.
